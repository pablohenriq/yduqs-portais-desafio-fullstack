# Monorepo

Um `monorepo` full-stack para **desafio técnico** da Yduqs

---

## Setup mínimo

Para instalar do dependencias de todos os pacotes:

```bash
pnpm install
```

Para inicializar a infra (Postgres):

```bash
pnpm dev:infra
```

Para executar as duas aplicações em modo de desenvolvimento:

```bash
pnpm dev
```
